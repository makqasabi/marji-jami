# مجلد ملفات الكتب (PDF)

تُحفظ هنا ملفات الكتب مرتَّبةً بحسب العلم: `pdfs/<اسم-العلم-بالشرطات>/<معرف-الكتاب>.pdf`
(مثال: `pdfs/شروح-الحديث/fath-al-bari.pdf`).

## مهم — حقوق النشر
- **الكتب الحرة** (التراث في الملك العام، أو ما تتيحه جهتُه مجانًا): يجوز تنزيلها وحفظها هنا.
- **المراجع المعاصرة المحمية**: لا تُنزَّل من مواقع غير مشروعة؛ تُقتنى قانونيًّا، وروابط شرائها مثبتة في صفحة
  [روابط التحميل](../docs/روابط-التحميل.md) بوسم 🛒.

روابط المصادر لكل كتاب في حقل `روابط_التحميل` داخل ملف البيانات، وتُجمع آليًا في `docs/روابط-التحميل.md`.

## ملاحظة git
ملفات الـPDF نفسها **مستثناة من git** (لكبر حجمها) — هذا المجلد وبنيته فقط محفوظان.
