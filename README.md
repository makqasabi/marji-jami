# المرجع الجامع لأمهات الكتب الإسلامية والعربية (مَرجِع جامِع)

قاعدة معرفية موثَّقة لأمهات الكتب في الحضارة العربية والإسلامية: الكتب التي هي «أمٌّ» في بابها، ذات الرصانة العلمية العالية، التي يُغني الواحد منها عن عشرات غيره في فنّه.

## ركائز المشروع الثلاث

1. **التراث الشرعي واللغوي**: علوم القرآن والحديث والفقه والعقيدة واللغة والأدب والتاريخ (مثل: فتح الباري، لسان العرب، تفسير الطبري، المغني لابن قدامة).
2. **العلوم الحضارية التراثية**: قمم الإنتاج العلمي العربي والإسلامي في الطب والرياضيات والفلك والبصريات والكيمياء والعمران (مثل: القانون لابن سينا، المناظر لابن الهيثم، مقدمة ابن خلدون) — تدخل لمرجعيتها التاريخية والحضارية لا بوصفها مراجع تطبيقية معاصرة.
3. **المراجع المعاصرة بالعربية**: الموسوعات والمعاجم والمراجع التخصصية الرصينة الصادرة بالعربية في العلوم الحديثة (مثل: الموسوعة العربية العالمية، المعجم الطبي الموحد، الوسيط للسنهوري).

## بنية المستودع

```
marji-jami/
├── README.md          ← هذا الملف
├── MANHAJ.md          ← معايير الإدخال والاستبعاد (المنهج العلمي)
├── CONTRIBUTING.md    ← كيفية إضافة كتاب أو تحديث بياناته
├── CHANGELOG.md       ← سجل المراحل المنجزة
├── STATUS.md          ← حالة العمل ونقطة الاستئناف
├── data/              ← مصدر الحقيقة: ملف YAML لكل كتاب
│   └── <العلم>/<معرف-الكتاب>.yaml
│   └── <العلم>/_تمهيد.md      ← تمهيد صفحة العلم (يُحرَّر يدويًا)
├── scripts/
│   ├── common.py      ← الثوابت المشتركة (قائمة العلوم، المحمّل)
│   ├── validate.py    ← التحقق من سلامة كل ملف وفق المخطط
│   └── build.py       ← توليد docs/ من data/
└── docs/              ← مخرجات مولّدة آليًا — لا تُحرَّر يدويًا
    ├── index.md
    ├── <العلم>.md
    ├── فهرس-المؤلفين.md
    ├── فهرس-القرون.md
    └── إحصاءات.md
```

## الاستخدام

```powershell
# التحقق من سلامة البيانات
python scripts/validate.py

# توليد الفهارس والصفحات
python scripts/build.py
```

(على هذا الجهاز: استخدم المسار الكامل لمفسّر بايثون 3.12.)

- ابدأ التصفح من [docs/index.md](docs/index.md) — الفهرس العام بروابط صفحات العلوم.
- كل صفحة علم تبدأ بتمهيد يعرّف بالعلم وطبقات كتبه، ثم جدول موجز، ثم بطاقات الكتب المفصلة.
- فهارس مساعدة: المؤلفون، القرون، الإحصاءات (وفيها قائمة الكتب الموسومة «يحتاج_تحقق»).

## ضمانات الصدق

- لا يُختلق اسم محقق ولا دار نشر ولا سنة طبع. كل بيان غير متيقَّن يوسم `حالة_التوثيق: يحتاج_تحقق` مع ملاحظة.
- نسبة الكتب الموسومة «يحتاج_تحقق» يجب ألا تتجاوز 15% (تُراقَب آليًا في إحصاءات.md).
- التفصيل الكامل في [MANHAJ.md](MANHAJ.md).
