# كيفية إضافة كتاب أو تحديث بياناته

## الخطوات

1. تأكد أولًا أن الكتاب يستوفي معايير [MANHAJ.md](MANHAJ.md) — وإلا فلا تضفه.
2. حدد العلم من القائمة في `scripts/common.py` (لا تخترع اسم علم جديد؛ إن احتجت علمًا جديدًا فعدّل القائمة أولًا في مراجعة مستقلة).
3. أنشئ ملفًا باسم `data/<العلم-بالشرطات>/<معرف-الكتاب>.yaml`:
   - المعرف بأحرف لاتينية صغيرة وشرطات (kebab-case)، مثل `fath-al-bari`.
   - يجب أن يطابق حقل `id` داخل الملف اسمَ الملف.
4. املأ الحقول وفق المخطط أدناه. لا تختلق أي بيان (انظر ضوابط الصدق في المنهج).
5. شغّل التحقق ثم التوليد:
   ```powershell
   python scripts/validate.py
   python scripts/build.py
   ```
6. راجع صفحة العلم المولدة في `docs/` ولا تحرر ملفات `docs/` يدويًا أبدًا.
7. أضف سطرًا في `CHANGELOG.md`.

## مخطط ملف الكتاب (الحقول المعلَّمة ★ إلزامية)

```yaml
id: fath-al-bari                      # ★ يطابق اسم الملف
العنوان: فتح الباري بشرح صحيح البخاري   # ★ العنوان الكامل الصحيح
المؤلف:                                # ★
  الاسم: أحمد بن علي بن حجر العسقلاني   # ★
  الوفاة_هجري: 852                     # ★ للأعلام المتوفين (رقم)
  الوفاة_ميلادي: 1449                  # ★ كذلك؛ للهيئات والأحياء تُحذف الوفيات ويُضاف "هيئة: نعم" أو "معاصر: نعم"
العلم: شروح الحديث                     # ★ من قائمة common.py حرفيًا
التصنيف_الفرعي: شروح صحيح البخاري       # اختياري
لماذا_هو_أم_في_بابه: >                  # ★ فقرة محكمة 80–150 كلمة
  ...
يغني_عن: []                            # قائمة كتب يقوم مقامها
يُقرأ_قبله: []                          # مداخل تمهيدية إن وجدت
أفضل_الطبعات:                          # ★ قائمة، الأولى هي المعتمدة
  - المحقق_أو_المعتني: ...
    الدار: ...                         # ★ داخل كل طبعة
    السنة: ...
    المجلدات: ...
    ملاحظة: لماذا هذه الطبعة معتمدة
التوافر_الرقمي:                        # ★
  المكتبة_الشاملة: نعم                  # نعم / لا / غير معروف
  منصات_أخرى: []
روابط_التحميل:                         # اختياري: مصادر تحميل الملف
  - المصدر: المكتبة الشاملة            # اسم الجهة/المنصة
    الرابط: https://shamela.ws/book/XXXX
    النوع: حر                          # حر (ملك عام/مجاني من جهته) أو شراء (محمي)
    # قاعدة صارمة: لا يُسجَّل رابط «حر» إلى موقع نسخ غير مشروع لكتاب محمي؛
    # المحمي يُسجَّل «شراء» برابط جهة بيع شرعية أو منصة الجهة الرسمية المجانية.
الشروح_والمختصرات_والذيول: []
مستوى_القارئ: متخصص                    # ★ مبتدئ / متوسط / متخصص (يجوز الجمع بـ /)
نوع_المرجعية: شرعية_لغوية              # ★ شرعية_لغوية / علمية_تراثية_حضارية / معاصرة
ملاحظات_نقدية: >                       # ما أُخذ على الكتاب بإنصاف؛ إلزامي للنوعين الثاني والثالث
  ...
حالة_التوثيق: موثّق                    # ★ موثّق / يحتاج_تحقق
مصادر_التوثيق: []                      # ★ ما اعتُمد عليه في التوثيق
آخر_تحديث: 2026-06-12                  # ★ تاريخ ميلادي
```

## تمهيد صفحة العلم

كل مجلد علم يحوي ملف `_تمهيد.md` (150–250 كلمة): تعريف بالعلم، طبقات كتبه، ترتيب القراءة المقترح، والتصريح بالفجوات إن وجدت. يدمجه `build.py` في رأس صفحة العلم.

## التحديث

عند تعديل أي بيانات: حدّث `آخر_تحديث`، وإن رفعت كتابًا من «يحتاج_تحقق» إلى «موثّق» فأثبت ما اعتمدت عليه في `مصادر_التوثيق`.
